◆ caesar()

int caesar ( ifstream * input_file,
ofstream * output_file,
offsets offset )

Функция реализующая шифр цезаря.

В качестве смещений принимаются offsets::lat, offsets::kir и offsets::num для смещений по латинице, кириллице и числам соответственно. При положительных значениях сдвиг вправо, при отрицательных - влево.

Аргументы
[in]input_fileФайл с исходным текстом
[out]output_fileСозданный файл с зашифрованным текстом
[in]offsetСтруктура задающая сдвиги
[offset::symbol_count_needed]== true повлечет запись в конец файла одного байта - количества символов в исходном файле.
[offset::checksum_needed]== true повлечет запись в конец файла одного байта - контрольной суммы.
[offset::checksum_time_dependent]== true повлечет зависимость контрольной суммы от времени.
